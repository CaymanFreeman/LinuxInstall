## Pop! OS (22.04 LTS)

### Get Script:
```
wget https://raw.githubusercontent.com/CaymanFreeman/LinuxInstall/main/install.sh-O install.sh
```

### Auto Run Script:
```
wget -q https://raw.githubusercontent.com/CaymanFreeman/LinuxInstall/main/install.sh -O install.sh
chmod +x install.sh
./install.sh
sudo rm install.sh
```
